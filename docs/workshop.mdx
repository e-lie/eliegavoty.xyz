# Ateliers de livecoding et de creative coding

Tous les mardi soirs j'anime un atelier de creative coding :

- 19h - 23h
- Au hackerspace le <a href="https://fuz.re">fuz</a>
- 57 rue Étienne Marcel à Montreuil.

On y code :

- de la **musique avec FoxDot** 
- des **visuels en utilisant hydra**
- et pleins d'autres choses.

Cet atelier est voué à être collaboratif avec différents intervenants (du cookie collective notamment) au fil du temps.

Les tutoriels regroupés ici ont été écrit ou traduits de l'anglais à l'occasion de ces ateliers.
